What this site is about

This site can help you to avoid writing the same VBA code every time.

vba helpers is an open source procedures / functions collection intended to solve general Excel programming tasks like: find the last row or create a worksheet or color the range and many others.

Thus, the logic of working with Excel VBA API is hidden, you just call ready-made procedures / functions as needed. This approach makes your code more readable and reliable. All the helpers supplied with description of parameters and example of usage so you can easily apply them in your macros.
